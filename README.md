# FESL-DRE: Federated Edge-Semantic Learning for Decentralized Resilient Evacuation

This repository contains the simulation code, synthetic dataset, evaluation results, and figures for the paper:

**Federated Edge-Semantic Learning for Decentralized and Resilient Indoor Evacuation under Dynamic Hazards**

## Overview

FESL-DRE is a decentralized evacuation framework that integrates:

- edge-based semantic reasoning,
- federated learning for human-centric behavior modeling,
- gossip-based decentralized coordination,
- adaptive hazard-aware evacuation routing.

The framework is evaluated using a synthetic graph-based indoor evacuation environment under dynamic hazards and node failure scenarios.
